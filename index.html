Git Hired

Git Hired consumes data from the GitHub Jobs API and presents software development jobs via interactive cards. Jobs can be searched by keyword and location. There's a dark mode that can be toggled on and off, with the mode being saved to Local Storage.

Tech Stack: ReactJS, React Redux, React Hooks, axios, thunk

Find a Brewery!

Find A Brewery consumes the Open Brewery API and allows the user to search for breweries in the United States by state and keyword. Brewery cards are interactive and take the user to the brewery website. There's a dark (beer) mode that can be toggled on and off, with the mode being saved to Local Storage.

Tech Stack: Tech Stack: ReactJS, React Redux, React Hooks, axios, thunk
